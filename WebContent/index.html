Instructions

  • use the Update button to request changes to the map
  • use the Back button on your browser to go back
  • to zoom in: mouse-drag a rectangle on the map
  • change the variable on the map with the "Map of" menu
  • constrain the cruises with controls below the map
  • the Property-Property Plot button plots data from the cruises on the map, optionally restricted by the zoom rectangle
  • the Table of Cruises button provides
    • "data" - download all the data from this cruise
    • "plot" - a property-property viewer of all data from this cruise
    • "map" - maps this cruise, omly
